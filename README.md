# plantilla-cv-css-html
Plantilla de CV hecha con html y css desde 0

Es mi primer proyecto publicado, estruturado con css flexbox. Tiene una función de imprimir con los details abiertos de javascript.
He practicado los aprendizajes de los cursos Html y css de midudev (midu.dev)

Lo he organizado como si fuese un Currículum de papel real, implementando las medidas de A4 al tamaño de la página.

He usado este artículo como referencia para la estructura: https://www.resufit.com/es/blog/estructura-de-curriculum-html-guia-completa-para-crear-un-cv-web-profesional/
